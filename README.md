# Cipher-tool
A python program containing tools for decrypting classical ciphers

## Ciphers supported:
* Caesar shift
* Substitution cipher
* Affine shift
* (Transposition cipher)
## Analysis tools:
* Length
* Playfair
* Frequency analysis
* (Index of coincidence)
* (Bigram frequencies)
## Text manipulation tools:
* Capitalise
* Lowercase
* Switch case
* Remove puntuation
* Remove spaces
* Reverse
